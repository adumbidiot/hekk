pub fn set_quick_edit_mode(active: bool) -> Result<()>
Expand description

Set the quick edit mode of the attached console.

Disabling it prevents hangs when writing if the user is interacting with the console, but the user is unable to copy text from it. A hang will crash the program as the program is left unable to deal with window messages. This cannot be easily worked around as it seems like the kernel call itself hangs for the entire duration of the user interaction. Good Job Microsoft!
